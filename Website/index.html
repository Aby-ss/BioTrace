BioTrace

BioTrace is an integrated system of camera traps and sound sensors that regularly provides data input to an AI processing hub which can detect animals and record their numbers.

Challenge Information

Find a way to improve the tracking of biodiversity in the wetlands. This involves using technology such as camera traps to identify animals, particularly at night. Propose a small project by determining the appropriate technology to use, where to install it, and how many cameras are needed.

In Addition the robust design, BioTrace is fitted with an intelligent Machine Learning Model which constantly detects and keeps track of the animals and their species located in the observable radius of the device.

Backened Software Details

The AI was first trained using real image data and multiple different iterations of the species and animals to make it's predictions as reliable as possible

The AI is able to detect the trained Data with ease and can successfully render clear images to the user via a wireless data transfer system which focuses on high resolution results

Recognising whether there are animals present in image / sound recording. Image recognition and sound recognition will have two large datasets, separate for animals / miscellaneous.

Determine the number of animals within the image and produce zoomed-in images of each animal. Determine if there are overlapping sounds and if overlap is present isolate different sounds.

Compare zoomed-in images to existing dataset of images of common animal species in the wetland. Then based on the animal's size, shape, and body temperature: record their species.
